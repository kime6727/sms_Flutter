## 功能说明

这是 SMS 接码平台的 Flutter Web 客户端，包含以下功能：

### 1. 用户认证
- 注册/登录（设备 ID 自动登录）
- 用户信息管理

### 2. 服务选择
- 浏览所有可用服务（WhatsApp、Telegram 等）
- 搜索服务
- 查看服务图标和名称

### 3. 国家选择
- 选择服务后查看可用国家
- 显示国家旗帜和价格
- 按价格排序

### 4. 购买流程
- 确认购买信息
- 扣除积分
- 创建订单

### 5. 订单管理
- 查看所有订单
- 订单详情
- 激活订单获取手机号
- 等待接收验证码
- 复制手机号和验证码
- 取消订单

### 6. 充值功能
- 查看充值套餐
- 首充双倍优惠提示
- 支付流程（Apple IAP）

### 7. 个人中心
- 查看余额和积分
- 交易历史
- 通知中心
- 设置（语言切换）
- 帮助和关于

## 技术栈

- Flutter Web
- Provider 状态管理
- Go Router 路由
- 国际化支持（中英文）
- Material Design 3
